# photo_similarity
画像の色の分布から類似度の計算

学部3年の時のC言語演習の授業で好きなものを作っても構わないと指示されたので作成。
ピクセルの階調値の分布から画像の類似度を計算する
(※カルバックライブラー情報量ではない)

この課題で実装したもの
* ハッシュテーブル
* PNM形式の画像ライブラリ
* BMP形式の画像ライブラリ
* 類似度計算ライブラリ
* コマンドラインオプションパーサ
